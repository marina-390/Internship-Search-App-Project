Accessibility Statement

Our commitment to inclusive design

Our Commitment

InternHub is committed to ensuring digital accessibility for all users, including those with disabilities. We strive to make our platform usable for everyone, including people with vision, hearing, mobility, and cognitive disabilities.

Web Accessibility Standards

We aim to comply with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards set by the W3C (World Wide Web Consortium).

Accessibility Features

InternHub includes the following accessibility features:

  • Keyboard Navigation: Full functionality available via keyboard
  • Screen Reader Support: Compatible with major screen reader software
  • Color Contrast: High contrast ratios for readability
  • Resizable Text: Ability to increase font size
  • Alternative Text: Descriptive alt text for images
  • Form Labels: Proper labeling of form fields
  • Focus Indicators: Clear visual indicators for keyboard focus

Accessibility Tools

We recommend using the following tools to enhance your experience:

  • NVDA: Free screen reader for Windows
  • JAWS: Popular screen reader for Windows
  • VoiceOver: Built-in screen reader for macOS and iOS
  • TalkBack: Built-in screen reader for Android
  • Browser Extensions: Tools like Lighthouse and axe DevTools

Accessibility Limitations

Despite our best efforts, some functionality may have limitations. We continuously work to improve and welcome feedback from users.

Report Accessibility Issues

If you encounter accessibility issues while using InternHub, please let us know. We take your feedback seriously and will work to address any problems.

Third-Party Content

Some content on our platform is provided by third parties. While we encourage all users to follow accessibility guidelines, we're not responsible for third-party content accessibility.

Continuous Improvement

We are committed to continuous improvement of accessibility. We regularly:

  • Test our platform with assistive technologies
  • Review user feedback and accessibility reports
  • Update our code to meet current standards
  • Train our team on accessibility best practices

Legal References

InternHub endeavors to be compliant with:

  • Americans with Disabilities Act (ADA)
  • Section 508 of the Rehabilitation Act
  • Web Content Accessibility Guidelines (WCAG) 2.1
